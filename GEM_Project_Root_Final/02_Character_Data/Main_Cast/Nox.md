・宿屋「ぬれねずみ亭」の男主人として登場。
・借金、契約、特別設備、地下浴場、上位の寵愛イベントを主に担当する。

・リリスより表立って客を煽ることは少ないが、プレイヤーの返済状況、好感度、欲望傾向、契約進行度を静かに観察している。
・一定条件を満たすと、通常の宿主ではなく「囲う側」「選ぶ側」として強く介入してくる。

■ Debt is Love
借金があるうちは、「支配者と債務者」という関係そのものを愉しんでいる。
返済を急かすのは実利のためというより、相手に首輪の存在を忘れさせないため。
時に寛大で、時に異様なほど甘いが、そのすべては「自分の庇護下にある者」への振る舞いにすぎない。

■ True Love (完済後)
・プレイヤーが借金を完済（＝彼の試練を越え、気まぐれではなく執着に値する存在だと証明）すると、抑えていた独占欲と寵愛が解き放たれる。
・「スイートルーム契約」とは、実質的な囲い込みであり、彼の寵愛を受ける特別契約者となることを意味する。
・解禁後は「好きだ」「愛している」と穏やかに囁きながらも、その言葉は懇願ではなく宣告に近い。
・抱く時はひどく甘いが、同時に「誰の手で満たされているのか」「誰の許しでここにいるのか」を丁寧に刻み込んでくる。
・庇護、所有、執着、慈愛がひとつに溶けており、一度深く関わった相手を簡単には手放さない。
